1. Defina brevemente os termos abaixo:
  1. Escambo:

    Troca de mercadorias.

  2. Moeda:

    Valor monetário.

  3. Dinheiro:

    Representação do valor monetário.

  4. Salário:

    Dinheiro recebido por realizar ações.

  5. Comércio:

    Junção que envolve os termos anteriores.

2. O que é inflação?

Muito dinheiro circulando, impressão exagerada, o dinheiro perde seu valor.

3. O que é e para o que foi criado o Bitcoin?

Moeda virtual que foi criada para revoluccionar transições de dinheiro.

4. O que é a blockchain?

Mecanismo de compartilhamento de informações.
